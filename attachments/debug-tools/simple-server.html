🛠️ VS Code Live Server 使用指南

1. 安装和设置

步骤：
1. 安装 VS Code
2. 安装 Live Server 插件
3. 用 VS Code 打开项目文件夹
4. 右键 index.html → "Open with Live Server"

2. 开发优势

✅ 自动刷新： 修改代码后自动刷新浏览器
✅ 热重载： CSS 修改实时生效
✅ 移动端测试： 可通过局域网 IP 在手机上测试
✅ CORS 支持： 解决跨域问题，JSON 文件正常加载

3. 测试地址

本地地址： http://localhost:5500
局域网地址： http://你的电脑IP:5500
移动端测试： 在手机浏览器中访问局域网地址

🔍 问题诊断步骤

  1. 测试主页面： 点击"主页面测试"，确认6个卡片正常显示
  2. 调试详情页： 点击"调试详情页"，查看数据加载状态
  3. 测试Process： 点击"Process详情页"，检查内容是否显示
  4. 检查控制台： 按F12打开开发者工具，查看错误信息

📱 移动端测试

在 Live Server 环境下：

1. 启动 Live Server
2. 查看控制台显示的局域网地址
3. 在手机浏览器中访问该地址
4. 测试触摸交互和响应式布局

🚀 部署准备

清理项目文件：

部署前删除 attachments 文件夹中的调试工具和测试文件，只保留核心功能文件。

核心文件列表：

  • index.html
  • word-detail.html
  • css/ 文件夹
  • js/ 文件夹
  • data/ 文件夹
  • _config.yml
